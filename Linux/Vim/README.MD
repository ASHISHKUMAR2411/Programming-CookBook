VIM 
=======================


How to exit Vim
---------------

1. Press Exit to enter **Normal Mode**

2. Press `:` to enter **command mode** 

3. Type `q` then Enter.


Commands Common
---------------


- Syntax `:syntax on`

- Number `:set number`

- quit : `:q`  quite & save `:wq`


Configuration File
------------------

1. Create a configuration file:

`vim .vimrc`
